# Overview

This is a simple console bookstore app to practice working with C# and its fundamentals. It showcases working with variables, expressions, conditionals, loops, functions, classes and data structures. This app serves as a memory refresher for core programming concepts and is a learning tool.

# Development Environment

IDE: Visual Studio Code
Programming Language: C#
.NET SDK: .NET 6.0 or later
Operating System: Windows 10/11
Recommended Extensions:
-C# for Visual Studio Code
-.NET CLI
